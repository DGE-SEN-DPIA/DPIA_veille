## ⚡ À retenir (importance 3)

### Claude Sonnet 5 : rattrapage d'un lancement majeur non capturé [NOUVEAU — non capturé dans notes 01/07 et 02/07]
- **Fait :** Le 30 juin 2026, simultanément à la levée des contrôles export sur Fable 5, Anthropic a lancé **Claude Sonnet 5** — son modèle le plus agentique à ce jour. Sonnet 5 devient immédiatement le **modèle par défaut pour les plans Free et Pro** (disponible aussi sur Max, Team, Enterprise, Claude Code et la Claude Platform). Capacités : raisonnement adaptatif activé par défaut (1M tokens de contexte, 128 000 tokens de sortie max), tool use étendu (navigateur, terminal, code), cyber-safeguards by default (bloquent en temps réel les usages cyber dangereux — même niveau que Opus 4.7 et 4.8). Sur le benchmark de travail de connaissance interne d'Anthropic, Sonnet 5 dépasse légèrement **Opus 4.8**. Tarif API introductif : **$2 / $10 par million de tokens** (entrée / sortie) jusqu'au 31 août 2026, puis $3 / $15 à partir du 1er septembre.
- **Pourquoi ça compte :** Ce lancement, éclipsé dans la couverture presse par le retour de Fable 5, est potentiellement le fait le plus structurant pour l'adoption enterprise à court terme. Sonnet 5 offre des performances proches d'Opus 4.8 à un prix proche de Claude Sonnet 4.6 — ce qui réduit drastiquement la justification économique de Fable 5 pour les cas d'usage agentiques courants. Pour les DSI publiques françaises : la disponibilité via Mistral API (si distribution croisée confirmée) ou via AWS SageMaker et Microsoft Foundry doit être vérifiée pour le déploiement souverain.
- **Sources :** Anthropic, « Introducing Claude Sonnet 5 », anthropic.com/news/claude-sonnet-5, 30/06/2026 ; TechCrunch, « Anthropic launches Claude Sonnet 5 as a cheaper way to run agents », 30/06/2026 ; Help Net Security, 01/07/2026 ; simonwillison.net, 30/06/2026.

### International (hors Europe)

**EO Trump — deadline 2 juillet passée, regard vers le 1er août [SUITE, dernière mention 02/07]**
La deadline des 30 jours de l'EO « Promoting Advanced AI Innovation and Security » (signé 02/06/2026) est expirée hier sans confirmation de publication du Treasury AI cybersecurity clearinghouse — deuxième livrable attendu, non confirmé publiquement. Le CISA BOD 26-04 reste le seul livrable daté confirmé (10/06). Prochain jalon impératif : **1er août 2026** — les secrétaires Treasury, NSA-DoD, CISA et NIST disposent de 60 jours pour livrer : (1) un cadre de benchmarking classifié pour les modèles frontière ; (2) un régime volontaire d'accès gouvernemental pré-release. Le cadre GPT-5.6 Sol (preview ~20 orgs depuis le 26/06) opère déjà selon ce modèle, avant que le texte formel soit publié.

### Europe

**AI Act Omnibus — publication JOUE toujours attendue [SUITE, dernière mention 02/07]**
La publication au Journal officiel de l'UE n'a pas encore été confirmée au 03/07 (vote Conseil 29/06, délai habituel 3-7 jours ouvrés). J-30 avant le 2 août (applicabilité Art. 50 transparence + dispositions GPAI). Consultation haute-risque ouverte jusqu'au 23 juillet.

**Chine — règlement anthropomorphique IA [SUITE, dernière mention 01/07] — J-12**
Les Interim Measures for AI Anthropomorphic Interactive Services (CAC + 4 ministères, adoptées 10/04/2026) entrent en vigueur le **15 juillet 2026**. Périmètre : services simulant traits de personnalité, modes de pensée et styles de communication de personnes réelles pour une interaction émotionnelle continue. Principales obligations : assessment de sécurité avant lancement (seuil : 1 M utilisateurs inscrits ou 100 K MAU), 7 catégories de contenus prohibés (notamment auto-harm, dépendance émotionnelle), interdiction absolue de virtual companion intime pour les mineurs, consentement parental obligatoire sous 14 ans. Premier règlement mondial contraignant sur l'IA émotionnelle — à surveiller pour les implications de conformité des acteurs européens opérant en Chine.

**DeepSeek V4 officiel — mi-juillet, double tarification [SUITE DeepSeek, 1ère mention registre 23/06]**
Le 30 juin, TechNode et Pandaily (sources chinoises primaires) ont confirmé : le lancement officiel de **DeepSeek V4** est planifié pour **mi-juillet 2026**. V4-Pro (1,6T paramètres, 49B actifs, 1M tokens contexte, modes Thinking/Non-Thinking) et V4-Flash (284B total/13B actifs). Tarifs off-peak : V4-Pro $0,435/$0,87 par M tokens (-75 % vs. preview) ; V4-Flash $0,14/$0,28. **Peak-hour pricing** (×2) : 9h-12h et 14h-18h heure de Beijing — première fois pour un LLM frontière chinois. **Deprecation des anciens noms API** (deepseek-chat, deepseek-reasoner) : **24 juillet 2026** à 15h59 UTC, sans période de grâce. Impact direct pour les organisations françaises qui s'appuient sur DeepSeek via API : migration obligatoire avant le 24/07.
Sources : TechNode, 30/06/2026 ; Pandaily, 30/06/2026 ; api-docs.deepseek.com.
